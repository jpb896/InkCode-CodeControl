# InkCode-CodeControl
This is the code control used in InkCode 2.1 and above. It is based on WinUIEdit, which is based on Scintilla.

I forked WinUIEdit so I could add some features that would benefit InkCode.

(InkCode 1.0, 1.1 and 2.0 still use the official, upstream WinUIEdit rather than my fork)
